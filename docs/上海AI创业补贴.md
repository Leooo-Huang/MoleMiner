- **金额**: 最高1500万元的开办费及办公用房补贴 / 3年每年最高100万元的房租补贴 / 最高50%的活动实际支出额补贴，单项活动最高300万元 / 最高600万元和300万元的专项扶持 / 最高200万元的研发费用投入补贴 / 最高50%的扶持 / 最高500万元的研发补贴 / 最高10万元的一次性奖励 / 最高100万元的带宽流量、精准定位数据服务、专业云平台服务等补贴 / 最高500万元的项目资助 / 最高50万元的利息补贴 / 最高2%的担保费或保险费率补贴 / 最高300万元改制及上市补贴
- **条件**: 注册满一年；员工超5人；带动产业数字化转型升级；研发工业芯片硬核新技术；建立产业链联盟；投资和引进上下游企业；联合高校、科研院所等；开展人工智能、大数据、物联网、云计算、网络安全、数字孪生等关键技术、共性技术攻关并应用；突破重大“卡脖子”技术并应用；通过上海技术交易所、国家技术转移东部中心等平台承接高校技术成果项目转化；加快高新技术成果产业化；牵头或参与数字化共性标准、关键技术标准制定；开展基于北斗、5G等新型信息基础设施的创新应用；对公共数据进行深度加工和增值使用；将创新成果在社区、园区、街区等落地示范应用；属于“重点支持在线新经济企业白名单”

- **要点**:

  - 支持研发成果转化，最高补贴500万元
  - 设立10亿元在线新经济母基金
  - 对重点支持企业贷款给予利息补贴，最高50万元
  - 支持企业改制上市，最高补贴300万元
