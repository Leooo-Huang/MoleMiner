# 深圳AI创业补贴申领行动计划

> 适用人: 海外计算机博士，回国AI创业，注册OPC
> 制定日期: 2026-03-30
> 预计收益: 第一年 130-200万，三年潜力 500-1500万

---

## 第一步：注册 OPC + 入驻龙岗 OPC 社区（本周完成）

### 1.1 注册一人公司（OPC）

- **做什么**: 在龙岗区注册一家"一人有限责任公司"，经营范围包含"人工智能技术服务、软件开发、信息技术咨询"
- **在哪办**: 深圳市市场监督管理局 → 开办企业一窗通
- **链接**: https://amr.sz.gov.cn/ → "开办企业一窗通"
- **费用**: 0元（深圳注册公司免费）
- **需要材料**:
  - 身份证
  - 深圳地址（可用 OPC 社区地址，先联系社区获取）
  - 公司名称（建议含"智能""科技"）
  - 经营范围
- **注意**: 不要用你的旧公司。旧公司注册 2 年无运营，财务空白，不符合大部分补贴条件

### 1.2 入驻龙岗 OPC 社区

- **做什么**: 申请入驻龙岗区首批 OPC 社区，获取免费办公空间
- **在哪申**: 龙岗区人工智能（机器人）署
- **链接**: https://www.lg.gov.cn/ （龙岗政府在线）
- **政策依据**: 《深圳市打造人工智能OPC创业生态引领地行动计划（2026—2027年）》深工信〔2026〕2号
- **政策原文**: https://www.sz.gov.cn/cn/xxgk/zfxxgj/tzgg/content/post_12602687.html
- **你能拿到**:
  - 免费办公空间（根据团队规模）
  - 免费 OpenClaw 部署服务
  - 3 个月免费算力
- **怎么做**: 联系龙岗区人工智能（机器人）署，询问首批 OPC 社区入驻流程。目前社区在南山、龙岗、福田、光明等区布局

## 第二步：申请入户补贴（注册后立即）

### 2.1 博士入户补贴 — 10万元

- **做什么**: 以海归博士身份申请深圳市人才引进入户补贴
- **金额**: 博士 10万元（一次性发放）
- **在哪申**: 深圳市人力资源和社会保障局
- **链接**: https://hrss.sz.gov.cn/ → 人才引进
- **政策依据**: 《深圳市打造人工智能OPC创业生态引领地行动计划》第三条
- **条件**:
  - 新引进落户深圳的博士
  - 在深圳缴纳社保
- **需要材料**:
  - 博士学位证书 + 学历认证（留学回国需做教育部学历认证）
  - 深圳社保证明
  - 银行账户
- **教育部学历认证**: http://renzheng.cscse.edu.cn/ （如果还没做，先做这一步）

### 2.2 安居住房补贴 — 约3万元

- **做什么**: 申请青年人才过渡性住房或安居补贴
- **金额**: 租金补贴 1250元/月（税后1000元）× 最长24个月 = 约3万元
- **或者**: 租住市场参考租金60%的过渡性住房，最长36个月
- **在哪申**: 深圳市住房和建设局
- **链接**: https://zjj.sz.gov.cn/
- **政策依据**: OPC行动计划第四条"提供安居住房支持"

---

## 第三步：申请创业资助（1个月内）

### 3.1 留学回国青年创业资助 — 最高100万元

- **做什么**: 以留学回国青年身份申请创业资助
- **金额**: 最高 100万元
- **在哪申**: 深圳市人力资源和社会保障局
- **链接**: https://hrss.sz.gov.cn/ → 创业补贴
- **政策依据**: OPC行动计划第五条"提供创业启动支持"
- **条件**:
  - 留学回国青年人才
  - 在深圳创办企业
- **需要材料**:
  - 留学回国人员证明
  - 教育部学历认证
  - 公司营业执照
  - 创业计划书

### 3.2 鲲鹏青年创新创业项目 — 最高5万元奖励 + 最高100万元研发资助

- **做什么**: 申请入库鲲鹏青年创新创业项目
- **金额**: 创新创业奖励 最高5万元 + 研发资助 最高100万元
- **在哪申**: 深圳市科技创新局
- **链接**: https://sticapply.sz.gov.cn/ （深圳市科技业务管理系统）
- **政策依据**: OPC行动计划第五条

### 3.3 个人创业担保贷款 — 最高60万元（财政贴息）

- **做什么**: 申请创业担保贷款，政府贴息
- **金额**: 最高 60万元个人 / 最高 500万元小微企业
- **在哪申**: 深圳市公共就业服务中心 → 合作银行
- **链接**: https://hrss.sz.gov.cn/ztfw/cjjy/cyfw/
- **条件**: 毕业5年内的青年人才创办初创型企业
- **注意**: 这是贷款不是补贴，但政府贴息 = 免息贷款

## 第四步：领取龙岗区专项补贴（入驻后）

### 4.1 OpenClaw 免费部署 + 数据服务

- **做什么**: 在 OPC 社区领取免费 OpenClaw 部署服务
- **省下**: 约 10-30万元
- **政策依据**: 《深圳市龙岗区支持OpenClaw&OPC发展的若干措施》（龙虾十条）
- **政策链接**: https://politics.gmw.cn/2026-03/09/content_38635629.htm
- **怎么做**: 入驻 OPC 社区后，社区会提供免费部署服务

### 4.2 算力补贴 — 3个月免费 + 年最高100万元

- **做什么**: 领取新入驻企业免费算力额度
- **金额**: 3个月免费算力 + 之后年度最高100万元算力补贴
- **在哪申**: 龙岗区人工智能（机器人）署
- **条件**: 经认定的 OPC 社区新入驻企业

### 4.3 龙虾盒子（AI NAS）补贴 — 市场价30%

- **做什么**: 购买 AI NAS 硬件，享受 30% 价格补贴
- **怎么算**: 如果龙虾盒子市场价 3万元，你只付 2.1万元

### 4.4 数据治理/标注服务补贴 — 50%

- **做什么**: 购买数据治理、标注等服务用于 OpenClaw 开发
- **金额**: 按费用的 50% 补贴

## 第五步：申请训力券/模型券/语料券（3-6个月后，有实际研发后）

### 5.1 训力券 — 最高1000万元

- **做什么**: 申请算力购买补贴
- **金额**: 智能算力合同金额的 50%，单企最高 1000万元
- **在哪申**: 深圳市科技创新局
- **链接**: https://sticapply.sz.gov.cn/
- **条件**: 年租用智能算力费用 ≥ 50万元
- **政策依据**: 《深圳市打造人工智能先锋城市的若干措施》深工信规〔2024〕13号

### 5.2 模型券 — 最高200万元

- **做什么**: 申请大模型 API 调用补贴
- **金额**: 采购合同额的 30%，最高 200万元
- **条件**: 依托国家网信办备案的生成式 AI 模型

### 5.3 语料券 — 最高200万元

- **做什么**: 申请语料数据采购补贴
- **金额**: 合同额的 40%，最高 200万元
- **条件**: 通过数据交易平台购买语料

---

## 第六步：申请种子基金/示范项目（有MVP后）

### 6.1 龙岗种子基金 — 最高1000万元股权投资

- **做什么**: 申请龙岗区 OPC 种子基金
- **金额**: 最高 1000万元（股权投资，不是白给）
- **条件**: 科技含量高、创新能力强的种子期 OPC 项目
- **政策依据**: 龙虾十条第七条

### 6.2 应用示范项目 — 最高400万元

- **做什么**: 申请龙岗区 AI 应用示范项目
- **金额**: 最高按项目实际投入的 50%，最高 400万元
- **条件**: 聚焦智能制造、智慧政务、智慧园区、智慧医疗等领域

### 6.3 OPC 黑客松/大赛奖金 — 最高50万元

- **做什么**: 参加龙岗区举办的 OPC 黑客松或创新创业大赛
- **金额**: 获奖最高 50万元
- **条件**: 在龙岗区主办的比赛中获奖

## ⚠️ 注意事项

1. **先做学历认证**：如果还没做教育部海外学历认证，这是所有海归补贴的前置条件
2. **不要用旧公司**：注册2年无运营的公司，财务空白，不符合大部分补贴要求
3. **政策时效**：龙虾十条目前是征求意见稿（2026年3月），正式版可能有调整
4. **OPC行动计划有效期至2027年底**：尽早入驻，越早拿越多
5. **部分补贴是事后奖补**：需要先花钱再申报补贴（如训力券），需要资金垫付
6. **同一项目不重复申报**：市级和区级同类补贴通常不能叠加，选金额大的申请
